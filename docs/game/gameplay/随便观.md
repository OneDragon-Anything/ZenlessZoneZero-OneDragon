# 随便观经营

澄辉坪「随便观」经营玩法:周期自动托管跑收益 + 多子玩法(游历 / 饮茶仙 / 制造坊 / 售卖铺 / 好物铺 / 邦巢 / 德丰大押)。**重 app** —— `SuibianTempleApp` 调度 7+ 子 op,每子 op「从入口进 → 操作 → 返回入口」。

## 入口

大世界 → `Transport(澄辉坪, 随便观)` → **interact(F) 狮耶**(传送点旁 NPC,`>`狮耶`<` 可交互态)→ 狮耶对话(「振兴随便观」)→ click 对话框推进 → **随便观-入口**。
- app `goto_suibian_temple`:Transport 后传送点就在狮耶交互范围内(**无 move,不走**;走了反而过头错过交互点);OCR 点交互提示文字(`['前往随便观','确认','领取收益']`)+ check 入口。

## 入口两态

- **自动托管中态**:中央「LV.X / 完成复兴 / 自动托管中」;游历 / 经营 tab **隐藏(锁)** —— 托管中不能手动派遣 / 制造。
- **手动态**:托管停止后,「游历」+「可派遣」/「经营」+「可制造」tab 显示 + 「暂停营业」。

## app 编排(`SuibianTempleApp` 节点链 + config 开关)

托管与子玩法**互斥**(托管中入口 tab 锁),按 `config.auto_manage_enabled` 分两支:

**开(默认)**:识别初始 → (传送 → interact 狮耶)→ `handle_auto_manage`(`SuibianTempleAutoManage`:停止托管领收益 → click「开始托管」**重启托管**)→ 返回大世界。**不跑子玩法**(托管跑收益)。

**关**:识别初始 → (传送 → interact 狮耶)→ `handle_auto_manage`(跳过,返「未开启自动托管」)→ 游历 → 饮茶仙(条件)→ 饮茶仙后再游历(条件)→ 制造坊 → 售卖铺 → 好物铺(条件)→ 邦巢(条件)→ 德丰大押(代码未开)→ 返回大世界。

**config 开关**(决定子玩法是否跑):
- `auto_manage_enabled`:开 → 只托管;关 → 跑子玩法链。
- `yum_cha_sin`:开 → 跑饮茶仙(且饮茶仙后再游历派遣);关 → 跳过(游历后直接制造)。
- `good_goods_purchase_enabled` / `boo_box_purchase_enabled`:好物铺 / 邦巢购买。
- `adventure_mission_1~4` / `adventure_duration`:游历副本 + 时长(默认科研院旧址系列 / 20小时)。

## 子玩法

- **游历**(`SuibianTempleAdventureSquad` claim+dispatch / `SuibianTempleAdventureDispatch`):收获(到期游历小队 → 可收获 → 确认 → 自动选择邦布)+ 派遣(选副本 → 选时间 → 邦布 → 派遣)。详见 [随便观.md](../screens/随便观.md) 游历部分。
- **制造坊**(`SuibianTempleCraft` / `SuibianTempleCraftDispatch`):制造台开工 + 邦布派驻。
- **售卖铺**(`SuibianTempleSalesStall`):货架售卖 + 拾遗。
- **饮茶仙**(`SuibianTempleYumChaSin`):定期采办委托(补材料 → 提交)。
- **好物铺**(`SuibianTempleGoodGoods`):经营购置商品(条件,无独立 screen_info)。
- **邦巢**(`SuibianTempleBooBox`):聘用 S 级邦布(条件)。
- **德丰大押**(`handle_pawnshop`):代码未开启(`round_success('未开启')` 占位)。
- **自动托管**(`SuibianTempleAutoManage`):停止托管领收益 + 重启托管(无独立 screen_info)。
