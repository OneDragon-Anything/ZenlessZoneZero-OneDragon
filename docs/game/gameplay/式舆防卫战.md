# 式舆防卫战(玩法)

绝区零周期战斗玩法(每周刷新),常驻。app `shiyu_defense`。画面见 [式舆防卫战.md](../screens/式舆防卫战.md)。

## 目标

挑战「剧变节点」(5 个节点,难度递增),通关获周期奖励(菲林 / 材料)。每周刷新。

## 入口与周期

- **入口**:快捷手册 → 作战 tab → 式舆防卫战「前往」(传送)。
- **周期**:每周刷新(周限倒计时,如「04 天 13 时」)。新一期官方可能允许从中间节点开始(如第 3 间,见 issue #2506)。

## 循环(一次完整挑战)

选关主界面 → 选节点 → 识别弱点 / 抗性 → 选配队 → 出战 → 战斗(自动)→ 战斗结束(下一防线 / 退出 / 撤退)→ 下一节点 → … → 全完成 → 领奖 → 返回。

## 机制

- **5 节点**:节点-01..05,难度递增。**节点 5 = 多间模式**(三间选择,3 间各自战斗,见 `式舆防卫战-三间选择`)。
- **剧变节点进度**:显示 X/Y(已完成 / 总数)。⚠️ OCR 常连写(`3/5`→`315`),app `choose_node_idx` 首尾数字法解析(current=digits[0], total=digits[-1],见 #2510)。
- **弱点 / 抗性**(2 阶段):每阶段 2 弱点 + 2 抗性,据此选克制配队(`shiyu_defense_team_utils.calc_teams`)。
- **编队**:预备编队(`ChoosePredefinedTeam`),按弱点配。
- **战斗**:`ShiyuDefenseBattle`(自动战斗,按配队 auto_battle 脚本)。
- **奖励**:全节点完成领奖(`全部领取`)。

## 自动化(app `shiyu_defense`)

节点链:传送 → 等待加载 → 选择节点(OCR 进度)→ 识别弱点并计算配队 → 选择配队 → 出战 → 自动战斗 → 下一节点 → 所有节点完成 → 领取奖励 → 关闭奖励 → 结束后返回。多间模式(节点 5)走 `multi_room_*` 节点(选房间 → 备战 → 出战 → 战斗 → 退出 → 循环)。
