# 式舆防卫战

绝区零周期玩法(每周刷新),常驻战斗挑战。app `shiyu_defense`。入口:大世界 → 快捷手册(作战 tab)→ 式舆防卫战「前往」→ 传送确认 → 选关主界面。

## 何时出现 + 状态流转

传送进 → 选关主界面 → [选节点] → 弱点/抗性 + 选配队 → 出战 → 战斗 → 下一节点 / 战斗结束 → 全完成 → 领奖 → 返回大世界。

**子态**(同 screen `式舆防卫战`,多子态共用 screen_info `shiyu_defense.yml`):

| 子态 | 入口 | 出口 |
|---|---|---|
| 选关主界面 | 传送进 / 战斗结束返回 | 点节点-XX → 弱点 |
| 弱点+抗性+选配队 | 选关后 | 点角色头像 → 选配队 |
| 战斗 | 出战(Deploy) | 敌人全灭 → 战斗结束 |
| 战斗结束 | 战斗胜利 | 下一防线 / 退出 / 撤退 |
| 领奖 | 全节点完成 | 全部领取 → 关闭 → 返回 |

**多间模式**(节点 5):点节点-05 → `式舆防卫战-三间选择`(选 3 间 → 各自战斗,见 `shiyu_defense_select_3.yml`)。

**剧变节点进度**:`剧变节点进度` area 显示 X/Y(如 3/5)。⚠️ **OCR 常连写为 `315`**(`/` 被识成 `1`),app `choose_node_idx` 用首尾数字法解析(`current=digits[0]`, `total=digits[-1]`,见 PR #2510 / issue #2506)。
