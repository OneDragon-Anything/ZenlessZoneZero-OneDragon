# 随便观(经营玩法)

澄辉坪「随便观」经营玩法:周期托管 + 多子玩法(游历 / 饮茶仙 / 制造坊 / 售卖铺 / 好物铺 / 邦巢 / 德丰大押)。**重 app** —— 主 app `SuibianTempleApp` 调度 7+ 子 op,每子 op「从入口进 → 操作 → 返回入口」独立流程。

## 入口 + 状态流转

大世界 → `Transport(澄辉坪, 随便观)` → OCR 点击「前往随便观」(建筑交互提示) → **随便观-入口** → 各子玩法 tab。

> ⚠️ 进随便观内部**不是 interact(F)**,是 OCR 点击「前往随便观」文本(app `goto_suibian_temple`:`round_by_ocr_and_click_by_priority(['前往随便观','确认','领取收益'])`)。interact 狮耶只会触发 incidental NPC 对话(见 [对话.md](对话.md))。

app 顺序流转(`SuibianTempleApp` `@operation_node` 链):
识别初始画面 → (传送 → 前往随便观) → **自动托管** → 游历 → 饮茶仙(条件) → 饮茶仙后再游历 → 制造坊 → 售卖铺 → 好物铺(条件) → 邦巢(条件) → 德丰大押(代码标未开启) → 返回大世界(`BackToNormalWorld`)。
